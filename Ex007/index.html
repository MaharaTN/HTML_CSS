Texto marcado

Para criar um texto marcado(marca texto), se usa a tag MARK.

E no outro parágrafo, temos outro texto marcado e para evitar de criar sempre um style para cada mark dentro da página é possível que se crie um seletor de uma tag, no caso a style.

Texto grande e pequeno

Estamos criando um texto grande utilizando big que é uma tag obsoleta e um texto pequeno nesse parágrafo. E porque o BIG caiu e SMALL não? É possível que pelo BIG não ter sentido haja vista que ele era utilizado apenas por aparência, ele entrou em desuso, sendo que o SMALL ele tem uma semântica, por ter um sentido de utilizar letras pequenas em determinadas situações, como as letras pequenas dos contratos (apenas uma possibilidade)

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.(semântico)

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.(semântico)

Existe também o sublinado com a tag U(não semântico)

Texto Sobrescrito

Serve para inserir coisas do tipo x20+3 Utilizando a tag SUP e que serve para cima

Texto Subscrito

para inserir coisas do tipo H20 Utilizando a tag SUB e que serve para baixo
